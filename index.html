About This Bot

This bot is a Telegram utility bot for tokusatsu related tasks such as fetching next episode preview video and downloading latest episode subtitle file.

Please note that this bot is currently only supporting operations for Kamen Rider series

Technology

Technologies used in this bot's development

  • Javascript

  • Node JS

  • Express JS

  • MongoDB

Additional Libraries

  • Cheerio

  • Mega JS

  • YTDL-Core

Features

  • Subtitle

    /izusubs

    Get latest episode subtitle file from Izusubs

  • Preview Video

    /nextpreview

    Get the preview video for the next episode

  • Preview Images

    /nextstill

    Get all preview images for the next episode from the official website

    More commands, tasks, and supported series are coming soon
